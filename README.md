# fish-chezmoi

[Chezmoi](https://github.com/twpayne/chezmoi) abbreviations

## 🚀 Installation

Using [fisher](https://github.com/jorgebucaran/fisher)

```bash
fisher install gazorby/fish-chezmoi
```

## 🔧 Usage

| Abbreviation | Command              |
| ------------ | -------------------- |
| cm           | chezmoi              |
| cma          | chezmoi add          |
| cmea         | chezmoi edit --apply |
| cmap         | chezmoi apply        |
| cmu          | chezmoi update       |
